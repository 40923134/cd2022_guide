Block UI Styler

Application -> Gateway -> More -> Block UI Styler

NXOpen_Getting_Started_2019.pdf 中 Chapter 14: Block-Based Dialogs 說明如何利用 Block UI Styler 應用程式建立 API 程式所需要的表單.

NX1953 Block UI Styler Documentation

利用 Block UI Styler 建立表單, 用來控制 block.prt 零件的長寬與高: block_ui.7z (NX2008 零件檔案)

操作方法: 以 NX2008 開啟 block.prt, 利用 Menu -> Tools -> Journal -> Play 執行 length_width_height.py, 待出現方塊長度、寬度與高度的設定表單後, 填入所需尺寸後, 按下 Ok, 零件即可依照設定修改尺寸. 此為利用 Block UI Styler 所製作的互動式零件參數設定. 若將表單移置網誌 Brython 前端, 則可依相同方式設定 Server 上的 NX2008 零組件檔案的設計參數.

OnEditOptionSelected Callback:

def OnEditOptionSelected(
    tree, node, columnId, selectedOptionID, 
    selectedOptionText, type):
    try:
        if type == NXOpen.BlockStyler.Tree.ControlType.ComboBox:
            if selectedOptionText == "RejectText":
                OnEditOptionSelected = 
                    NXOpen.BlockStyler.Tree.EditControlOption.Reject
            else:
                OnEditOptionSelected = 
                    NXOpen.BlockStyler.Tree.EditControlOption.Accept
        elif type == NXOpen.BlockStyler.Tree.ControlType.ListBox:
            if selectedOptionText == "RejectText":
                OnEditOptionSelected = 
                    NXOpen.BlockStyler.Tree.EditControlOption.Reject
            else:
                OnEditOptionSelected = 
                    NXOpen.BlockStyler.Tree.EditControlOption.Accept
    except Exception as ex:
        # ---- Enter the exception handling code here. -----
        pass

OnMenuCallback:

class MenuID:
    AddNode = 0
    DeleteNode = 1
    SubMenuItem1 = 2
    SubMenu1 = 3
def OnContextMenuCallback(self, tree, node, columnID):
    try:
        menu = tree.CreateMenu()
        SubMenu1 = tree.CreateMenu()
        menu.AddMenuItem(MenuID.AddNode, "Add Node")
        menu.AddSeperator()
        menu.AddMenuItem(MenuID.DeleteNode, "Delete Node")
        # Sub menu
        SubMenu1.AddMenuItem(MenuID.SubMenuItem1, "Sub Menu Item 1")
        # Set submenu
        menu.SetSubMenu(MenuID.SubMenu1, SubMenu1)
        # Set menu on the tree
        tree.SetMenu(menu)
        # Dispose the menu only after setting the menu on the tree
        menu.Dispose()
        SubMenu1.Dispose()
    except Exception as ex:
        # ---- Enter the exception handling code here. -----
        self.theUI.NXMessageBox.Show(
            "Block Styler", 
            NXOpen.NXMessageBox.DialogType.Error, 
            str(ex))
